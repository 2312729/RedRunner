## Mô tả
Bài thực hành Lab 01 môn **Game 2D Development with Unity**.
Khám phá dự án game RedRunner - Platformer 2D mã nguồn mở.

## Các thay đổi đã thực hiện
1. **Thay đổi tốc độ chạy**: 8 → 15
2. **Thay đổi lực nhảy**: 12 → 24 (Nhân vật nhảy cao hơn để vượt địa hình)
3. **Thay đổi trọng lực (Gravity Scale)**: 
   - 0.5: Nhân vật bay bổng, rơi chậm.
   - 3.0: Nhân vật rơi rất nhanh, cảm giác nặng nề.
4. **Thêm Coin**: Đã kéo thả Prefab Coin vào Scene tại khu vực bắt đầu game để kiểm tra tính năng thu thập.

## Kiến thức đã học được
1. Cách sử dụng Inspector để tinh chỉnh vật lý (Rigidbody 2D).
2. Phân biệt va chạm vật lý (Spike) và va chạm xuyên thấu (Water/Coin).
3. Quy trình làm việc chuyên nghiệp với Git (Add, Commit, Push).
4. Phân tích cấu trúc mã nguồn dự án mẫu
5. Cách quản lý dự án thông qua file README.md.
